Georgia Absentee Voter Data

    

Tool for browsing the Georgia Absentee Voter Data using the Exhibit Javascript library.
    https://systemsplanet.github.io/

A couple of things pop out from this tool:

   - Clicking "Ballot was Undelivered" shows valid BallotReturnDates!
   - Ballot Ids are being reused. Is that ok, or indication of fakery?



Unfortunately Exhibit uses Jquery and is way too slow for this much data.

But I'm hoping this tool will motivate younger brighter people than me to duplicate what I've done using something faster and lighter.

Recommend using chrome to open the 1st link which is data from 118.

After a few moments you'll be able to click to drill down on any of the fields.

Each click updates the GUI which is really slow, so be patient.
